
HOWTO make:

  Just type `make`.

Usage:
  ./myls [-lavrchR]
    [-l]        Display long info. 输出详细信息
    [-a]        Display all files.
    [-v]        Debug info. 输出调试信息
    [-r]        Reverse sort.
    [-c]        Case SeNsiTivE sort. 大小写敏感排序
    [-R]        Recursive list.
    [-h]        This help message.
